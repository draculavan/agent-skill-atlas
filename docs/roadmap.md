# Roadmap

Agent Skill Atlas is an early open-source release. The roadmap favors privacy,
compatibility, and real user feedback over feature volume.

## Now

- Collect sanitized skill folder layouts from Codex, Claude, Cursor, and custom
  local agent users.
- Improve taxonomy presets and keyword matching.
- Review privacy defaults and redaction behavior.
- Add a short demo GIF or video for the README.

## Next

- Add fixture packs for common skill layouts.
- Improve plugin summary mode.
- Add optional schema validation for generated `atlas.json`.
- Add import/export examples for team-safe sanitized demos.

## Later

- Consider a lightweight hosted demo that uses only bundled demo data.
- Explore team governance features for internal skill registries.
- Explore package publishing after install/update behavior is stable.

## Non-Goals For Now

- Uploading private skill data.
- Executing skills from the atlas UI.
- Replacing local agent routing.
- Publishing to npm before the install/update path is better tested.
